AI使用指南：从入门到进阶的四个层级

MEM研究生「AI技术革命」课程笔记 + 实操指南
2026年6月28日 · 阅读约15分钟

今天「AI技术革命」课上，老师提了一个问题：你们平时怎么用AI？

大部分同学的回答是"用豆包聊天"、"问GPT问题"、"让Kimi帮我写东西"。这没有错，但这只是冰山一角。

老师把AI的使用分成了四个层级，从"消费者"到"驾驭者"，每一层的效率和可能性都是指数级的跃迁。

这篇文章把课上的知识点整理出来，附带手把手的搭建教程，帮你从Level 1往上走。

🎯 四个层级总览

Level 1

网页/APP 端使用 AI

代表：豆包、ChatGPT、Kimi、文心一言、通义千问

特征：打开网页或APP，像聊天一样使用AI。门槛最低，功能也最受限。你只能用别人给你的界面，无法自定义，无法自动化。

类比：就像用计算器——能算数，但你没法让计算器自动帮你做报表。

Level 2

在 IDE 中使用 AI

代表：Cursor、GitHub Copilot、Windsurf、VS Code + AI插件

特征：AI嵌入到编程工具中，能看懂你的代码上下文，自动补全、重构、Debug。你需要会基本的编程环境操作。

类比：从计算器升级到Excel——能处理更复杂的任务，能复用模板。

Level 3

AI 编程代理（Coding Agent）

代表：Claude Code、Codex（OpenAI）、Gemini CLI

特征：AI不再只是辅助，而是自主执行。你给它一个目标，它自己读代码、改代码、运行测试、修复错误。需要命令行操作能力。

类比：从自己干活升级到有个助手——你出想法，它出活。

Level 4

AI 工作流平台（Agent Platform）

代表：OpenClaw、Hermes

特征：AI具备持久记忆、多工具调用、自动化工作流。它能读你的文件、发消息、管理日程、操作浏览器、调用各种API。是真正的"AI员工"。

类比：从一个助手升级到一个团队——有记忆，能协作，能主动干活。

📊 层级对比

维度 L1 网页端 L2 IDE L3 Agent L4 平台
门槛 零门槛 会装软件 会命令行 会配置服务
能做的事 聊天、问答 写代码、Debug 自主开发项目 自动化一切
效率提升 1.5x 3-5x 10x 50x+
需要的技能 打字 基础编程 命令行 + 编程 系统配置能力

🔧 从 Level 1 到 Level 3：手把手搭建

前置准备：获取 API Key

从Level 2开始，你需要API Key。简单理解：网页端是"用别人的车"，API Key是"拿到车钥匙，想开去哪就去哪"。

💡 什么是 API Key？
API Key 就像一把钥匙。你拿着这把钥匙，任何支持这个AI的软件都能调用它。网页端只能在一个界面用，API Key 则到处都能用。

推荐：小米 MiMo（国内直连，无需翻墙）

如果你还没有"翻墙"能力，小米 MiMo 是最佳起点：

  1. 打开 https://mimo.xiaomi.com
  2. 注册/登录小米账号
  3. 进入开发者中心，申请 API Key
  4. 复制你的 Key，保存好（不要泄露给别人）
✅ MiMo 优势：国内直连、中文友好、有免费额度、兼容 OpenAI 格式（大部分工具都能直接用）

其他 API 来源（部分需要翻墙）

服务 是否需要翻墙 费用 特点
小米 MiMo ❌ 不需要 有免费额度 国内首选，兼容性强
DeepSeek ❌ 不需要 很便宜 国内，性价比极高
Claude（Anthropic） ✅ 需要 $20/月起 编程能力最强
ChatGPT（OpenAI） ✅ 需要 $20/月起 生态最完善

🚀 Level 2：在 IDE 中使用 AI

推荐工具：Cursor（最容易上手的 AI 编程 IDE）

  1. 下载 Cursor：https://cursor.sh
  2. 安装后打开，它长得像 VS Code（因为基于 VS Code 改的）
  3. 在设置中填入你的 API Key（或用 Cursor 自带的额度）
  4. 打开任意代码文件，Ctrl+K 让 AI 帮你写/改代码
⚠️ 注意：Cursor 本身需要翻墙下载。如果暂时不会翻墙，可以先用国内替代方案 Trae（字节跳动出品，国内可下载）。

🚀 Level 3：AI 编程代理

代表工具：Claude Code

Claude Code 不是一个软件界面，而是在终端（命令行）里运行的AI。你跟它说"帮我做一个网站"，它会自己创建文件、写代码、运行、调试，全程自主完成。

安装步骤

# 1. 先安装 Node.js（去 https://nodejs.org 下载安装）

# 2. 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 3. 设置 API Key
export ANTHROPIC_API_KEY=你的key

# 4. 启动
claude

启动后，你就像聊天一样告诉它要做什么。它会自动操作你的文件系统，写代码、运行命令、修复错误。

💡 Codex（OpenAI） 类似，但目前更偏向云端运行，需要 ChatGPT Plus/Pro 订阅。对新手来说 Claude Code 更直观。

🚀 Level 4：AI 工作流平台

代表：OpenClaw

OpenClaw 不只是一个编程工具，而是一个AI运行平台。它可以：

  • 连接你的微信、飞书、Discord 等消息平台
  • 读写你的文件、管理日程、发邮件
  • 操作浏览器、调用各种 API
  • 拥有持久记忆——它记得你之前说过什么
  • 自动化定时任务，比如每天早上给你发天气和日程

简单说，Level 3 是"AI帮你写代码"，Level 4 是"AI帮你处理一切"。

# 安装 OpenClaw
npm install -g openclaw

# 初始化配置
openclaw init

# 启动
openclaw start
⚠️ Level 4 的门槛较高，建议先把 Level 2 和 Level 3 玩熟了再尝试。但一旦搭建成功，效率提升是质变级别的。

📌 行动建议

如果你现在是 Level 1（大多数人），建议路径：

  1. 今天：注册小米 MiMo，拿到 API Key（5分钟搞定）
  2. 这周：下载 Cursor 或 Trae，体验 AI 辅助编程（30分钟上手）
  3. 这个月：尝试 Claude Code，用 AI 自主完成一个小项目
  4. 未来：搭建 OpenClaw，让 AI 成为你的工作流核心

别停在 Level 1

AI 不只是聊天工具。层级每上一层，你的效率就是别人的十倍。

💡 实战案例：30分钟，从零到一个上线的网站

下面是今天课堂上真实发生的对话。我（一个MEM研究生，非计算机背景）和AI协作，从零搭建了一个公开网站。整个过程没有手写一行代码。

第一步：明确目标（1分钟）

👤 我：我想做个网页挂在公网上面，你知道怎么操作吗？

要点：不需要告诉AI用什么技术，只需要说清楚你要什么。AI会帮你选方案。

第二步：确认方案（2分钟）

🤖 AI：推荐 GitHub Pages，免费，不需要服务器。
🤖 AI：需要三样东西：网页文件 + 托管 + 域名。
👤 我：好，那我们开始。

要点：AI给方案时会列出选项，你只需要选一个方向，不用纠结技术细节。

第三步：AI全自动执行（10分钟）

👤 我：你不能直接操作github吗？
🤖 AI：可以，我帮你登录GitHub → 创建仓库 → 上传文件 → 开启Pages
👤 我：（只需要在浏览器里点一次授权，其他全自动）

关键动作：

  • AI 自动执行了 gh auth login（GitHub命令行登录）
  • AI 自动创建了仓库 Ludwig-Emergence.github.io
  • AI 自动编写了HTML页面并上传
  • AI 自动开启了GitHub Pages服务
  • 我全程只做了一件事：在浏览器里点了一下"授权"

第四步：迭代内容（15分钟）

👤 我：写一篇科普文章，关于AI使用的四个层级...
👤 我：（描述了课程内容和想要的文章结构）
🤖 AI：写好了，排版完成，已发布上线。
👤 我：再加一个实战案例部分。
🤖 AI：好的，改完了，已上线。

要点：你只需要描述"要什么"，AI负责"怎么做"。不满意就说，AI立刻改。

复盘：我做了什么？AI做了什么？

我（人类）做的 AI 做的
说"我想做个网站" 选择技术方案
注册GitHub账号 登录、创建仓库
在浏览器点了一次授权 编写HTML、CSS代码
描述文章内容和结构 排版、上传、部署
说"再加一个案例" 修改代码、重新发布
总计：约5句话 总计：数百行代码 + 全套部署
✅ 这就是 Level 3/4 的工作方式：你负责方向和决策，AI负责执行。你的每一句话，都是在"指挥"一个程序员帮你干活。

❓ 常见问题

Q：我不会翻墙，能用 Level 2/3 吗？

A：可以。小米 MiMo 和 DeepSeek 都是国内直连。Cursor 的替代品 Trae 也可以国内下载。Claude Code 本身用的是 API，只要你的 API 来源能连上就行。

Q：API Key 会不会很贵？

A：MiMo 有免费额度，DeepSeek 的价格极低（几分钱一次调用）。比 ChatGPT Plus 的 $20/月便宜得多。

Q：我完全不会编程，应该从哪开始？

A：先用 Level 1 的 AI 学编程基础。让豆包/GPT 教你写 Python，边学边用。然后再往 Level 2 走。AI 本身就是最好的编程老师。

Q：这些工具安全吗？API Key 泄露了怎么办？

A：API Key 不要公开分享（不要发到朋友圈、群里）。如果不小心泄露了，去对应平台重新生成一个就行，旧的会失效。
