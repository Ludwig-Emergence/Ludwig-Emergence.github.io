动手搭建：获取你的 AI 能力

认知到位了，现在动手。这一章带你完成最关键的一步：拿到 API Key，让 AI 从"聊天工具"变成"你的工作引擎"。

🔑 第一站：申请 MiMo API Key

为什么推荐小米 MiMo？

  • 国内直连——不需要特殊网络，注册就能用
  • 中文能力最强——小米自研，中文理解一流
  • 兼容 OpenAI 格式——几乎所有工具都能直接接入
  • 有免费额度——先体验，不花钱
1

打开 MiMo 开发者平台

访问 https://mimo.mi.com，你会看到如下界面：

点击右上角 「控制台」 进入登录页面。用小米账号登录，没有小米账号的用手机号注册一个，2分钟搞定。

2

充值余额

MiMo 没有免费额度，需要先充值才能使用。但价格非常便宜：

模型输入价格输出价格
MiMo-V2.5（推荐）¥1 / 百万Token¥2 / 百万Token
MiMo-V2.5-Pro¥3 / 百万Token¥6 / 百万Token

建议先充 ¥30，够你用很久。一次普通对话大约消耗几千 Token，30 块钱可以对话上万次。

在控制台 → 「费用中心」 → 「充值」，支持支付宝。

3

创建 API Key

登录后，在控制台找到 「API Key 管理」 → 「创建 API Key」

系统会生成一串以 sk- 开头的密钥。立刻复制保存，这个 Key 只显示一次。

💡 花费预估：MiMo-V2.5 的价格是 ¥1/百万输入Token、¥2/百万输出Token。一次普通对话大约几千 Token，¥30 足够你体验上千次对话。不用担心一下花完。
4

选择你的模型

MiMo 提供多个模型，新手推荐 MiMo-V2.5（标准版），性价比最高。

🛠️ 第二站：配置你的 AI 工具

拿到 API Key 后，接下来把它接入一个工具，让 AI 真正为你工作。

方案 A：Cursor（最容易上手的 AI 编程 IDE）

1

下载安装 Cursor

访问 https://cursor.sh 下载安装。它长得像 VS Code，因为就是基于 VS Code 改的。

⚠️ 需要特殊网络环境下载。如果没有特殊网络环境，用方案 B（Trae）。
2

配置 API Key

打开 Cursor → Ctrl+,（或 Cmd+,）打开设置 → 搜索 OpenAI API Key → 粘贴你的 MiMo Key

在 Base URL 填入：https://api.xiaomimimo.com/v1

3

开始使用

打开任意文件，按 Ctrl+K，输入你想要的效果，AI 会自动写代码。

💡 试试这个提示词

帮我写一个 Python 脚本，读取当前目录下所有 CSV 文件，合并成一个 Excel 文件，第一列为文件名来源

方案 B：Trae（国内可下载，无需特殊网络）

1

下载 Trae

访问 https://trae.ai（字节跳动出品），国内直接下载。安装后打开，自带 7 天免费体验额度，不需要配置任何 API Key 就能直接用。

2

免费额度用完后，接入 MiMo

7 天免费额度体验完后，如果想继续用，可以把 MiMo 接进去。

设置 → 模型管理 → 添加自定义模型。

⚠️ MiMo 有两种 Key，别选错了：

类型前缀来源用在哪
API Keysk-控制台 → API Key 管理按量付费（充多少用多少）
Token Plan Keytp-控制台 → Token Plan订阅套餐（包月/包年）

在 Trae 里：
• 如果选了 mimo-plan，必须填 tp- 开头的 Key
• 如果选 通用 OpenAI 兼容，填 sk- 开头的 Key，API 地址填 https://api.xiaomimimo.com/v1

3

开始使用

打开任意文件，直接用自然语言告诉 Trae 你想要什么效果，AI 会自动写代码。

💡 试试这个提示词

帮我写一个 Python 脚本，读取当前目录下所有 CSV 文件，合并成一个 Excel 文件，第一列为文件名来源

💻 插播：什么是终端？5分钟入门

接下来的方案 C 和 D 都需要用到"终端"。如果你从没用过，花 5 分钟学一下，不难。如果你只打算用方案 A 或 B，可以跳过这一节。

终端（Terminal）就是一个用文字操作电脑的窗口。你平时用鼠标点来点去，终端是用打字来操作。听起来吓人，其实就几个命令。

第一步：打开终端

1

Mac 用户

按 Command + 空格，输入 Terminal（或"终端"），回车。会弹出一个黑底白字的窗口。

2

Windows 用户

按 Win + R，输入 cmd，回车。或者在开始菜单搜索"命令提示符"。

第二步：认识终端界面

打开后你会看到类似这样的一行字：

用户名@电脑名 ~ % ← 这叫"提示符"，意思是"我准备好了，请输入命令"

你只需要在光标闪烁的地方打字，然后按回车执行。

第三步：记住这 5 个命令就够了

命令作用类比
pwd显示你当前在哪个文件夹看地图定位
ls列出当前文件夹里的文件打开文件柜看里面有什么
cd 文件夹名进入某个文件夹走进一个房间
cd ..返回上一级文件夹退出房间
clear清空屏幕擦黑板
💡 试试看：打开终端，输入 pwd 然后回车。会显示你当前的位置，类似 /Users/你的名字。这就是你的"家目录"。

第四步：理解"复制粘贴"的区别

终端里的复制粘贴快捷键和平时不一样：

操作MacWindows
复制Command + CCtrl + C（但这个是取消！右键复制）
粘贴Command + VCtrl + V 或右键 → 粘贴
⚠️ Windows 用户注意：Ctrl+C 在终端里是取消/中断命令，不是复制。要复制文字，选中后右键 → 复制。

第五步：什么是"安装"软件？

终端里安装软件，就是输入一行命令，电脑自动帮你下载和配置。比如：

# 这行命令的意思是：用 npm（Node.js 的工具）安装 claude-code 这个软件 npm install -g @anthropic-ai/claude-code

你不需要理解每个词的意思。复制这行命令，粘贴到终端，回车，等它跑完就行。

💡 看到一堆文字在滚动？正常的，那是电脑在告诉你它在干什么。等到又出现提示符（用户名@电脑名 ~ %），说明执行完了。

📚 方案 C：了解 Claude Code 与 Codex（Level 3）

⚠️ 国内用户请注意：Claude Code 和 Codex 在国内基本无法使用。

原因：
• 注册需要海外手机号——国内手机号无法注册 Anthropic（Claude）和 OpenAI（Codex）账号
• 支付需要海外信用卡——不支持支付宝、微信支付，需要 Visa/Mastercard 外币卡
• API 调用需要特殊网络环境——即使有了账号，直连也会被拦截

本节仅作科普了解，可以跳过。如果你有海外手机号和外币卡，才需要看下面的内容。

Claude Code 和 Codex 是目前最强的 AI 编程代理（Level 3 工具），了解一下它们的能力边界是有价值的：

Claude Code（Anthropic）

在终端里运行的 AI 编程代理。你给它一个目标，它自己读代码、改代码、运行测试、修复错误。编程能力公认最强，但月费 $20 起 + 需要特殊网络环境。

安装方法（需要先装 Node.js → https://nodejs.org）：

# 安装 npm install -g @anthropic-ai/claude-code # 设置 Key 并启动 export ANTHROPIC_API_KEY=你的Key claude
⚠️ 实测发现：Claude Code 无法直接接入 MiMo。虽然 MiMo API 兼容 Anthropic 协议格式，但 Claude Code 内部硬编码了 Claude 模型名，不接受 MiMo 的模型名。想用 Level 3 工具，推荐下面的 MiMo Code。

Codex（OpenAI）

OpenAI 的编程代理，目前更偏向云端运行，需要 ChatGPT Pro 订阅（$200/月）。门槛更高。

✅ 国内 Level 3 首选：MiMo Code

MiMo Code 是小米出品的 AI 编程代理，国内直连，一行命令安装，有免费模式：

# 一键安装 curl -fsSL https://mimo.xiaomi.com/install | bash # 验证 mimo --version # 启动（默认免费模式，不需要 Key） mimo

• MiMo Auto（免费）：默认模式，限免使用 MiMo-V2.5，无需登录
• Token Plan：订阅用户，用 tp- Key 登录
• API 模式：按量付费，用 sk- Key 登录

🔒 方案 D：OpenClaw（Level 4，进阶用户）

⚠️ OpenClaw 门槛较低，但有重大隐私风险，请认真阅读后再决定是否使用。

OpenClaw 是一个开源的 AI 工作流平台，能把 AI 变成一个有记忆、能操作各种工具的"AI 员工"。它的安装比 Claude Code 简单，但隐私风险也更大。

为什么说门槛低？

  • 开源免费，npm install -g openclaw 一行命令安装
  • 支持接入国内模型（MiMo、DeepSeek），不需要特殊网络
  • 配置简单，有引导流程

⚠️ 隐私风险——这才是重点

OpenClaw 不是普通聊天工具，它是一个深度介入你数字生活的平台。一旦接入，它可能拥有以下权限：

能力意味着什么风险等级
读写你的文件能看到你的文档、代码、笔记🔴 高
接入微信/飞书能读取你的聊天记录、联系人🔴 高
操作浏览器能访问你登录过的网站🔴 高
管理日程/邮件能看到你的日程和邮件内容🟡 中
调用外部 API能以你的名义发送请求🟡 中
持久记忆你之前说过的所有内容都会被记住🟡 中
🔴 核心风险：
• API Key 泄露——如果配置不当，你的 Key 可能被第三方获取
• 数据外传——AI 的对话内容会发送到模型提供商的服务器（国外模型发到海外）
• 过度授权——一旦给了文件/消息权限，AI 能看到的东西远超你的想象
• 提示词注入——恶意内容可能通过 AI 操控你的系统

建议：不要在存有敏感数据（银行信息、身份证、公司机密）的电脑上使用。如果要用，建议在独立的虚拟机或专用电脑上运行。

如果决定尝试

# 安装 npm install -g openclaw # 初始化（会引导你配置模型和 API Key） openclaw init # 启动 openclaw start

官方文档：https://docs.openclaw.ai
